# Quantum Device Simulator — Documentation

Welcome to the **Quantum Device Simulator** documentation.

## Overview

This application helps you:

- **Simulate devices** — Create virtual devices that output data over Serial (PTY) or TCP.
- **Connect to devices** — Use the Connections page to connect to any device (simulated or real) via Serial or TCP and view live data.
- **Visualize data** — Monitor streams with real-time charts, statistics, and CSV export.

## Quick start

1. Go to **Device Simulator** and create a device (e.g. Sensor, Serial).
2. Power on the device and note the Serial path or TCP port.
3. Open **Connections**, add a connection using that path or host:port, then connect and open **Monitor** to see the stream and charts.
